# Cardiovascular Risk: Lifestyle, Blood Pressure & Global Hypertension Context

**Final Exam Data Science Project**

This project investigates how lifestyle and clinical variables relate to cardiovascular disease risk. It combines:

1. **Kaggle Cardiovascular Disease Dataset** - patient-level data with age, gender, height, weight, blood pressure, cholesterol, glucose, smoking, alcohol intake, physical activity, and the cardiovascular disease target.
2. **WHO / Our World in Data hypertension prevalence dataset** - country-level hypertension prevalence among adults aged 30-79, used as an external population-level context source.

## Research question

Can lifestyle and basic clinical indicators explain and predict cardiovascular disease risk, and how does individual-level blood pressure risk compare with global hypertension patterns?

## Hypotheses

- H1: Higher BMI and higher blood pressure are associated with higher cardiovascular disease risk.
- H2: Physical activity is associated with lower cardiovascular disease risk.
- H3: A model using several risk factors predicts cardiovascular disease better than a single-factor baseline.
- H4: Patient-level hypertension signals are consistent with the broader global hypertension burden.

## Data setup

### Kaggle dataset

Download from Kaggle:

https://www.kaggle.com/datasets/sulianova/cardiovascular-disease-dataset

Place the file here:

```text
data/raw/cardio_train.csv
```

Expected separator is `;`. The notebook automatically handles it.

### WHO / OWID dataset

The notebook can download this automatically from Our World in Data:

```text
https://ourworldindata.org/grapher/hypertension-adults-30-79.csv
```

If internet access is unavailable, download it manually and place it here:

```text
data/raw/hypertension-adults-30-79.csv
```

## How to run

```bash
pip install -r requirements.txt
jupyter notebook notebooks/cardiovascular_risk_pro_project.ipynb
```

Run all cells from top to bottom.
